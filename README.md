# Old DVD Screensaver

The animation of old DVD Screensaver created on React Native using Reanimated 2 library.

# Requirements

- Node >= 11.10.1
- Expo >= v45.0
- npm (>8.1.0)
- React Native >= 0.70

# Installation

run 
```bash
npm install
```
command on your terminal.

After all packages installed. Run

```bash
npm start
```
to run the expo server. Then press "i" to run it on IOS.
